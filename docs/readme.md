# Documentation - Heroic-Legacy

Welcome to the project documentation hub. This directory contains all technical and design documentation for the Heroic-Legacy project.

## 📂 Directory Structure

* **/specs** : Design documents, raid strategies, and class mechanics breakdown.
* **/database** : SQL schemas, custom tables documentation, and DB design notes.
* **/scripts** : Technical documentation regarding C++ scripts and SmartScripts logic.
* **/guides** : Installation guides, development environment setup, and troubleshooting.
* **/archives** : Historical data, deprecated patches, and old changelogs.

---

## 🛠️ How to contribute to the docs

1. **Keep it simple:** Use Markdown for all documentation.
2. **Version control:** When making major changes to the project design, update the corresponding document here to reflect the state of the `main` branch.
3. **Link it:** If you add a guide, reference it in the main root `README.md` to ensure it's easily accessible.
